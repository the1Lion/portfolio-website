# Project 01 — A Developer-First Design System

## Overview
An opinionated, accessible component library built for Next.js — dark-mode native, motion-first, and zero-config.

## Problem
Most design systems are either too opinionated (MUI) or too bare (shadcn). There is no dark-mode-first, motion-ready library with zero lock-in.

## Approach
Build on top of Radix UI primitives, expose design tokens as CSS custom properties, and ship animations via framer-motion.
